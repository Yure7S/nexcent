Repositório dedicado à resolução de um teste técnico de desenvolvimento Frontend. O projeto consiste no desenvolvimento de uma landing page institucional responsiva, validando competências em arquitetura de componentes, estilização moderna e fidelidade de layout.

🚀 **Deploy:** [Acesse o projeto ao vivo](https://nexcent-neon-ten.vercel.app)

## 📋 Escopo do Teste
O objetivo era avaliar a capacidade de entrega de código limpo e escalável sob demandas específicas:
1. **Fidelidade Visual:** Implementação precisa do design proposto para a marca Nexcent.
2. **Responsividade:** Layout adaptável e otimizado para qualquer dispositivo.
3. **Escolha Tecnológica:** Uso do ecossistema React/Next.js combinado com Tailwind CSS para uma estilização rápida e eficiente.
4. **Deploy Automatizado:** Configuração de pipeline contínua via Vercel.

## 🛠️ Tecnologias
* **Framework:** Next.js
* **Estilização:** Tailwind CSS
* **Deploy:** Vercel
